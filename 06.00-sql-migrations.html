migrationهای SQL

در بخش بعدی کتاب، بیایید به چیزی کمی concreteتر و ملموس‌تر برگردیم و قدم‌هایی برداریم تا یک table به نام movies در database مربوط به greenlight بسازیم.

برای انجام این کار، می‌توانستیم دوباره به‌سادگی از ابزار psql استفاده کنیم و statement لازم CREATE TABLE را روی database اجرا کنیم. اما به جای آن، بررسی می‌کنیم چطور از SQL migrationها برای ساخت table استفاده کنیم، و به طور کلی‌تر چطور تغییرات schema مربوط به database را در طول پروژه مدیریت کنیم.

درباره این موارد یاد می‌گیرید:

  • اصل‌های سطح بالا پشت SQL migrationها و اینکه چرا مفید هستند.

  • چطور از ابزار command-line مربوط به migrate برای مدیریت programmatic تغییرات schema در database استفاده کنید.
