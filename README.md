# lets-go-further-fa.ir

جلد دوم کتاب *Let's Go*

نسخهٔ فارسی کتاب **Let's Go Further** اثر [Alex Edwards](https://www.alexedwards.net/)؛ ادامهٔ مستقیم کتاب محبوب [*Let's Go*](https://lets-go.alexedwards.net/) که الگوهای پیشرفته‌تر برای ساخت APIها و برنامه‌های وب با Go را قدم‌به‌قدم آموزش می‌دهد.

در طول کتاب، برنامهٔ **Greenlight** را از صفر می‌سازیم: یک API مبتنی بر JSON برای دریافت و مدیریت اطلاعات فیلم‌ها (شبیه [Open Movie Database API](http://www.omdbapi.com/)). از مسیریابی RESTful و کار با JSON شروع می‌کنیم، سپس به PostgreSQL، عملیات CRUD، فیلتر و صفحه‌بندی، محدودیت نرخ، احراز هویت، ارسال ایمیل، و در نهایت استقرار روی سرور لینوکسی می‌رسیم.

این کتاب به‌صورت خاص برای کسانی طراحی شده که [*Let's Go*](https://lets-go.alexedwards.net/) را خوانده‌اند و می‌خواهند گام بعدی را بردارند. اگر هنوز جلد اول را نخوانده‌اید، پیشنهاد می‌کنم ابتدا نسخهٔ فارسی آن را ببینید:

- وب‌سایت: [lets-go-fa.ir](https://lets-go-fa.ir/)
- مخزن GitHub: [RaziPour1993/lets-go-fa.ir](https://github.com/RaziPour1993/lets-go-fa.ir)

## درباره این ترجمه

پس از اتمام ترجمهٔ [*Let's Go*](https://lets-go-fa.ir/)، طبیعی بود که سراغ جلد دوم هم بروم. همان روشی را ادامه دادم: ترجمهٔ اولیه با کمک هوش مصنوعی، و بعد بازخوانی و روان‌سازی دستی هر بخش. متن و سورس این ترجمه در گیت‌هاب منتشر شده است.

اگر وسط مطالعه به غلط املایی، اصطلاح نامفهوم، یا جمله‌ای که روان نیست برخوردید، حتماً به من بگویید — از طریق Issue یا Pull Request در [مخزن GitHub](https://github.com/RaziPour1993/lets-go-further-fa.ir).
